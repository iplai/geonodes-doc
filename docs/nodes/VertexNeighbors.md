## Initialization

```python
from geonodes import nodes
node = nodes.VertexNeighbors(label=None, node_color=None)
```



## Arguments


### Node label

- label : Geometry node display label (default=None)
- node_color : Geometry node color (default=None)

## Output sockets

- vertex_count : Integer
- face_count : Integer
