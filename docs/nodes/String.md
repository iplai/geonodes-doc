## Initialization

```python
from geonodes import nodes
node = nodes.String(string='', label=None, node_color=None)
```



## Arguments


### Parameters

- string : str (default = '')

### Node label

- label : Geometry node display label (default=None)
- node_color : Geometry node color (default=None)

## Output sockets

- string : String
