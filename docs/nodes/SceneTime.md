
# Node SceneTime

Node *Scene Time* ()

Args:
  label: The node label
  
Provides the *seconds* and *frame* value for animation.

The :attr:`Tree.scene` property maintains an instance of this node:

.. code-block:: python

  time = tree.scene.seconds
  frame_number = tree.scene.frame
