## Initialization

```python
from geonodes import nodes
node = nodes.CombineXyz(x=None, y=None, z=None, label=None, node_color=None)
```



## Arguments


### Input sockets

- x : Float

### Node label

- label : Geometry node display label (default=None)

## Output sockets
