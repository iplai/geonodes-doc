## Initialization

```python
from geonodes import nodes
node = nodes.InstanceRotation(label=None, node_color=None)
```



## Arguments


### Node label

- label : Geometry node display label (default=None)
- node_color : Geometry node color (default=None)

## Output sockets

- rotation : Vector
