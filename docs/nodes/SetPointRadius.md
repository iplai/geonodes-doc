## Initialization

```python
from geonodes import nodes
node = nodes.SetPointRadius(points=None, selection=None, radius=None, label=None, node_color=None)
```

## Arguments


### Input sockets

- points : Points

### Node label

- label : Geometry node display label (default=None)

## Output sockets
